# Getting Started with Vite

This project uses [Vite](https://vitejs.dev/) with React and TypeScript.

## Available Scripts

In the project directory, you can run:

### `npm start` / `npm run dev`

Runs the app in development mode.
Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

The page will hot-reload if you make edits.

### `npm test`

Runs the test suite once using [Vitest](https://vitest.dev/).

### `npm run test:watch`

Launches Vitest in interactive watch mode.

### `npm run build`

Builds the app for production to the `dist` folder.
It compiles TypeScript and bundles with Vite, optimizing for best performance.

The build is minified and filenames include content hashes.
Your app is ready to be deployed!

### `npm run preview`

Serves the production build locally for preview before deployment.
